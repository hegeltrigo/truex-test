# Truex Test

This is a tes for Truextend Company

## Getting Started

install git https://git-scm.com/download/linux

```
$ git clone git@github.com:hegeltrigo/truex-test.git
$ cd truex-test
```

### Prerequisites

rvm https://rvm.io/rvm/install

### Installing

```
$ rake db:drop db:create db:migrate 
$ rails s
```

## Running the tests

```
$ rake db:drop db:create db:migrate
$ rspec
```
